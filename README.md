# Repositório do SQUAD3

## Projeto 
Página Web Responsiva realizada na intenção de auxiliar na organização de filas para jogos de vídeo game.

Tecnologias usadas: 
- HTML5
- CSS3
- Javascript 
- PHP
- AJAX
- Banco de dados (Firebase)
